🎯 Alpha Plugin 使用说明

  1. 点击浏览器工具栏中的 Alpha Plugin 图标
  2. 在弹出窗口中点击"开始选择区域"
  3. 在此页面上拖拽选择任意区域（比如下面的卡片或表格）
  4. 添加描述并点击"抓取数据"
  5. 打开控制台（F12）查看输出结果
